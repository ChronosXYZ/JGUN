## Intro
A realtime, decentralized, offline-first, mutable graph protocol to sync the Internet.

## Requirements
* JRE/JDK >= 11

## Building
1. Clone repo:
```bash
$ git clone https://github.com/ChronosX88/JGUN
$ cd JGUN
```
2. Compile it:
```bash
./gradlew build
```
